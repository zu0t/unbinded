TLDR

Features

Smooth

Smooth transitions. Great animations. High performance. Instant input.

Easy to configure

Live reloading config. Easy plain-text format. Sensible defaults. Great documentation.

Dynamic tiling

Automatic tiling that just works. Supports multiple fine-tuneable layouts.

Tearing support Socket-based IPC Window groups Special workspaces
Global shortcuts for apps Touchpad gestures

Unlock full power

Get the latest features Linux offers. Have full control over your workflow by customizing and extending it how you want.
